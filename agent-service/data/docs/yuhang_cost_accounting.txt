文档 ID：Doc-YH-IND-COST-003
适用场景：储备方案成本测算、成本合理性校验、投资收益评估
一、余杭工业地块储备一级成本大类（法定可计入成本项目）
第一类：土地取得直接补偿费（占总成本 60%-75%）
1. 集体土地征收类费用
征地区片综合地价（土地补偿 + 安置补助）：余杭片区均价区间 28-52 万元 / 亩；
地上附着物补偿费：厂房、苗木、鱼塘、构筑物、迁坟补偿，据实核算；
被征地农民社会保障专项资金：余杭标准约 3.8-4.5 万元 / 亩；
青苗补偿费：1000-1500 元 / 亩；
耕地占补平衡指标费：余杭现行指标成本 9-12 万元 / 亩；
新增建设用地有偿使用费：约 14 元 /㎡（折合 0.93 万元 / 亩）。
2. 存量国有工业用地收购费用
土地使用权收购评估价款；
原有建（构）筑物拆除补偿、设备搬迁费；
司法处置地块拍卖价款、诉讼费、执行费。
第二类：前期开发整理成本（七通一平，占总成本 15%-25%）
场地平整土方工程：4-8 万元 / 亩（依地形高差浮动）；
园区基础设施配套：道路、给排水、电力、燃气、通信管网综合 12-20 万元 / 亩；
配套绿化、照明、围挡工程；
工程监理、造价咨询、施工图设计费（工程总造价 3%-5%）。
第三类：专项评估、测绘、行政服务类间接费用（占总成本 3%-6%）
地块现状测绘、勘测定界：150-200 元 / 亩；
四大风险专项评估（土壤、地质、矿产、文物）：单地块合计 8-15 万元；
土地不动产评估、征地风险稳评：5-10 万元 / 宗；
规划方案编制、报批专家评审费；
征地公证、听证、法律服务费用。
第四类：税费、融资财务成本
契税（收购存量土地按评估价 3% 征收）、印花税；
土地储备专项债券 / 银行贷款利息：按融资周期、利率据实计入；
贷款担保、发行中介服务费。
第五类：管护、预备类其他成本
储备期地块围挡、安保、扬尘治理管护费；
不可预见费：按前四类总成本合计的 5%-8% 计提（余杭统一标准）；
项目审计、清算专项费用。
二、余杭成本核算禁止列支项目（评估方案成本缺陷判定标准）
做地主体日常办公、人员薪酬等行政经费；
招商宣传、展会营销费用；
超出法定标准的额外补偿、奖励；
出让后政府产业扶持、企业税收返还资金。
三、余杭成本收益平衡测算通用模型参数
工业用地出让底价参考：余杭各板块工业出让均价 120-220 万元 / 亩（经济开发区、良渚新城偏高，百丈、鸬鸟等西部镇街偏低）；
做地主体合理利润上限：开发整理总成本 10% 以内（市级统一管控标准）杭州市规划和自然资源局；
资金周期基准：从立项到出让平均周期 24 个月，融资成本按年化 3.2%-4.0% 测算。
