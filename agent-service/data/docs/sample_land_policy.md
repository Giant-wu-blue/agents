# 浙江省工业用地储备政策(样例文档)

根据《浙江省土地储备管理办法》第十二条,工业用地储备需满足产业规划符合性、土地利用总体规划一致性两项基本要求。储备机构应当对拟储备地块开展前期调查,核实权属、现状用途与规划用途。

《浙江省工业项目建设用地控制指标》第五条规定,工业用地容积率原则上不低于1.0,鼓励建设多层标准厂房以提高土地集约利用水平。对于高新技术产业,容积率可适当提高。

杭州市余杭区工业用地以装备制造、生物医药为主导产业。该区域地块周边交通配套完善,临近高速出入口,适宜发展先进制造业。区域内已储备工业用地的平均楼面地价约为每平方米800元。

宁波市工业用地以临港工业、新材料为特色。北仑区依托港口优势,重点发展石化、装备制造产业。宁波工业用地储备成本受港口区位影响,核心区域楼面地价高于一般区域。

《土地储备资金财务管理办法》第八条要求,储备土地的成本核算应包括取得成本、前期开发成本、资金成本三部分。资金成本按实际占用天数和融资利率计算。
